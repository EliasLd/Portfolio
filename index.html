Mes projets

DUINOSHOOT

DUINOSHOOT est un petit jeu type space invader qui fonctionne avec un microcontrôleur Atmega328p sur une carte Arduino Nano. Il est jouable à l'aide de trois boutons poussoir et d'un écran oled de 128x64 pixels.

Le jeu est entièrement codé en C++ en utilisant des classes ainsi que les principes basiques d'évolution et d'interaction de différents acteurs dans un espace.

Voir le projet sur Github

Mots-glissés

Jeu de Mots-glissés programmé entièrement en C# avec une interface console. Utilise tous les principes de la programmation orientée objet. Le programme peut lire/sauvgarder des plateaux de jeux dans des fichiers .txt et .csv (Excel) grâce à l'utilsation de la classe Stream (System IO).

Le Jeu se base sur un dictionnaire de Mots (.txt). Développement d'algorithmes récursifs pour la rechecrche de mots sur le plateau, dans le dictionnaire et pour le tri rapide du dictionnaire.

Voir le projet sur Github

ESP-MEDIA

[ EN DEVELOPPEMENT ]
Il s'agit d'un système d'applications fonctionnant sur une carte
ESP-WROOM-32 (STM32) pourvue d'un écran tactile utilisant le protocole SPI. Entièrement codé en C/C++ avec PlatformIO, le système propose diverses applications et jeux utilisablent avec un stylet pour plus de précision.

Le projet s'orientera ensuite vers le web grâce à la configuration d'un Web Server sur l'ESP-32.
